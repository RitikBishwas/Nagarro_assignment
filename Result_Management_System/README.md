# Result Management System

A simple web application for teachers to add student results and view all results.

## Features
- Add student results (Roll No, Name, Date of Birth, Score)
- View all student results in a table
- Form validation and error messages
- Responsive UI using Bootstrap

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Bootstrap


## Requirements
- All fields are required and validated
- UI matches the provided mockup

## Bonus
- Bootstrap used for layout and forms
